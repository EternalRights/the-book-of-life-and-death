# 西游生死簿
&emsp;&emsp;基于Spring Boot的西游生死簿，灵感来源《西游记》，立意新颖，旨在打造有创意的Java后端项目。

## 前言
&emsp;&emsp;该项目正处于发展阶段，此项目是作者本人秉持着弘扬中华优秀传统文化的精神以及技术使生活充满乐趣的想法创造而来。  
&emsp;&emsp;此项目现阶段立项于中国四大名著之一《西游记》中的生死簿灵感，后续预计将会接入其余的四大名著，使得该项目的趣味性更上一层楼。
